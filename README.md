# otaku_vehicleshop

## Requirements

To run otaku_vehicleshop you will need to be using the following resources;

- es_extended
- mysql-async

## Download & Installation

- Import otaku_vehicleshop.sql in your database
- Add this in your server.cfg: "ensure otaku_vehicleshop"
